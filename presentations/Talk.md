# Title
Applying Software Engineering Practices to Preparing Training Courses

## Abstract
Learn how a team of four instructors leveraged GitHub's collaborative features
to systematically improve a Python data analysis course using software engineering best practices.
By implementing workflows such as issue tracking, pull requests and structured code review processes,
we transformed course development into an agile, iterative process.
This approach proved successful in our recent teaching and sets the foundation for future course iterations
where student feedback will be systematically incorporated to create more student-centred training materials,
demonstrating how modern development practices can revolutionise educational content development.
